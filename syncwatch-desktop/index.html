Syncwatch

v0.5.8

Download
Syncs the position and pause state of the video player between connected peers to watch videos together with friends.
Works with magnet links and starts playback instantly. Other supported video sources can be found here. Requires mpv and optionally youtube-dl to be installed on the system. Windows installer includes both.

Installers for MacOS and Linux are planned. Meanwhile the application can be used on these platform with the following:

git clone https://github.com/utkuce/syncwatch-desktop.git
npm install && npm start
